parametersR Documentation

Set parameters for a synthetic population

Description

The parameters used by the poptimesimul programs are set in this file.

Details

The global parameters used when creating a population are set in the file ‘parameters.R’. This file is sourced by the main program makepop.R. The default values of the parameters are in parentheses.

  • M0: average number of people per cluster (400)

  • H: number of strata (10)

  • C: a vector of length H indicating the number of clusters for each stratum (75, 75, 75, 75, 80, 80, 80, 75, 75, 100)

  • L: the number of months of data (120); the cases L=1 and L>1 are different

  • age-sex probabilities based on U.S. census data

  • employment and unemployment ratios by age-sex, based on Canadian survey data

  • education level: 1 = no high school diploma, ..., 4 = university degree, based on Canadian survey data

  • meaninc: overall mean annual income in thousands of dollars (50)

  • meaniccvar: overall mean for an income-like variable (50)

  • transition probabilities for change from employed to unemployed, etc. (based on Canadian survey data; only used if L>1)

Value

This program defines many variables that are then used by makepop.R.
