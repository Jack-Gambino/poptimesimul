poptimesimulR Documentation

Generate a synthetic longitudinal population useful for sample survey methods research and education

Description

poptimesimul generates a synthetic population with enough structure to make it possible for survey statisticians and students to use the population to (1) illustrate existing sampling and estimation methods and (2) try out new methods in a realistic setting. The population includes some basic static variables (age, sex, education level) and both discrete (labour force status) and continuous (income) analysis variables that change over time. The population evolves over many months in a way that is sufficiently realistic for the above two goals.

Details

The main program is makepop.R, which creates an initial population (a snapshot of the population at month 1) and then optionally makes the population evolve over L months (L=120 by default). Users who are not interested in longitudinal data can set L=1. Conversely, users who need many monthly snapshots of the population should set L to a value greater than 1. By default, L=120, i.e., ten years of monthly data. If the user wants L>120, then some changes are required since the program depends on an auxiliary file containing real data and the one in the package has 120 months of data.

Value

The output of makepop.R when L=1 is an .RData file containing a population with characteristics age, sex, education, labour force status and income. The population is grouped into clusters which are themselves grouped into strata. The output of makepop.R when L>1 is a population file that is extended over L months; see makepop for more information about how the data is stored.
