makemonthR Documentation

Make the population age by one month

Description

Given data for month t-1, use transition probabilities to produce data for month t

Usage

makemonth(month, e2e, e2u, u2e, u2u, n2e, n2u,
                 emp, unemp, inc)

Arguments

month

a value from 2 to L

e2e, e2u, u2e, u2u, n2e, n2u

vectors of transition probabilities for each person in the population (employment-to-unemployment, etc.)

emp, unemp, inc

population vectors of employment and unemployment status and income for month t-1

Details

makemonth() is called by makepop.R, once for each month, where month = 2, 3, ..., L. Each call produces a column in each of the matrices empmat, unempmat and incmat.

Value

Updated columns in the matrices empmat, unempmat and incmat as described in makepop.
